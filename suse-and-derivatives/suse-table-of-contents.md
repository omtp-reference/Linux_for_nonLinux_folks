## SUSE and Derivatives

SUSE is one of the oldest Linux distributions still in active development — founded in
Germany in 1992. It has a reputation for being extremely stable, enterprise-ready, and
for having some of the best system administration tooling in the Linux world.

Most people will encounter SUSE in one of two flavors: **OpenSUSE** (free, for home and
developer use) or **SUSE Linux Enterprise** (paid, for business servers and workstations).

### The Short Version

| Distro                    | Release Model                   | Best For                           |
| ------------------------- | ------------------------------- | ---------------------------------- |
| **OpenSUSE Leap**         | Point releases (~annually)      | Desktop users who want stability   |
| **OpenSUSE Tumbleweed**   | Rolling release                 | Users who want the latest software |
| **SUSE Linux Enterprise** | Long-term stable + paid support | Businesses and servers             |

If you're helping a friend who uses SUSE, it's almost certainly either **Leap** or
**Tumbleweed** — check which one they have with:

```bash
cat /etc/os-release
```
